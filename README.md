# View-Jobs
- 채용 사이트를 저장하고 관리
- webView 통해 빠르고 쉽게 채용 사이트의 공고를 확인 하기 위한 앱

# 이 앱을 만들게 된 이유
- 잡코리아/자소설 등에서 Android 나 안드로이드 검색을 하면 잡다한 것들이 나와서 찾기가 어려움
- 잡코리아/자소설 등 취업사이트에 등록을 안하는 기업들도 많아서 채용 회사 사이트에 들어가야 하는 불편함이 존재

## Stack & Skill (스택 기술)
- Kotlin & Android
- Room, WebView, ViewBinding
